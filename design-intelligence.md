# Design Intelligence Skill — Vibe Design Process

## Step 1: Hero Asset
- Go to spline.design → Discover → Community
- Find a 3D asset that matches the brand vibe
- Export as vanilla.js web content
- Integrate using the Spline skill

## Step 2: Font Selection
Reference sites for font inspiration:
- fontsinuse.com (see real-world font usage)
- typewolf.com (font recommendations)
- fonts.google.com (free fonts)
- Adobe Fonts (premium)
Method: screenshot a font style you like → drop it in AntiGravity → ask to match the font

## Step 3: UI Component Sniping
- Go to 21st.dev → search for components (buttons, cards, testimonials, forms)
- Go to codepen.io → search trending for interactive elements
- Copy the component URL or prompt → paste into AntiGravity
- Ask AntiGravity to integrate it into the website

## Step 4: Image Generation
- Use weevi.ai for mood board → image generation
- Drop in reference images + color palette + prompt
- Use Nano Banana Pro model for best results
- Download at minimum 2K resolution
- Remove background in Canva if needed

## Step 5: Color Palette
- Screenshot your current website
- Drop into Canva → use color picker to extract hex codes
- Apply consistent palette across all elements

## Design Rules for Built By Borch:
- Theme: Ultra-dark (#1a1a2e) with gold accent (#D4A853)
- Font: Modern, sleek, professional
- NO lime green, NO bright colors
- Hero image should be the ONLY main visual element per viewport
- Mobile responsive always
- Danish language for all UI text

Always read this skill before doing any design work on the platform.
