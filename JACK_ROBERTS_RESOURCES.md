# Jack Roberts AntiGravity Resources

This document serves as a central reference for all tools, skills, and workflows recommended by Jack Roberts for building elite websites with AntiGravity.

## 🚀 The BLAST Framework
The core workflow for all projects built by Jack Roberts.

1.  **Blueprint**: Initialize project memory using `task.md` and high-level design tokens.
2.  **Link**: Connect "Universal Remotes" (MCPs).
3.  **Architect**: Parallel build of frontend and backend logic.
4.  **Style ("Vibe Design")**: Apply high-end UI/UX using Spline and picked components.
5.  **Trigger**: Rapid deployment and automated triggers.

## 🛠 MCP Configurations
Add these to your `mcp_config.json` for full power.

### Vercel MCP (Official)
```json
"vercel": {
  "command": "npx",
  "args": ["-y", "@vercel/mcp-server"],
  "env": {
    "VERCEL_API_KEY": "<YOUR_VERCEL_TOKEN>"
  }
}
```

### Supabase MCP
```json
"supabase": {
  "command": "npx",
  "args": ["-y", "@supabase/mcp-server-supabase@latest", "--access-token", "<YOUR_SUPABASE_TOKEN>"],
  "env": {}
}
```
